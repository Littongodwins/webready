# WebReady Demo Template

This is a clean, high-conversion one-page website template for Ugandan SMEs.

## Features
- Hero section with clear CTA (WhatsApp)
- About Us section
- Services/Menu section with pricing
- Customer Reviews/Testimonials
- Google Maps integration
- Contact details and hours
- Mobile-responsive design using Tailwind CSS

## Dynamic Data
The template is designed to be populated from a JSON file.
See `src/data/business.json` for an example and `src/data/schema.json` for the data structure.

## How to use
1. Update `src/data/business.json` with the business details.
2. Run `npm install` and `npm run dev` to see the site.
3. Run `npm run build` to generate the static files.
